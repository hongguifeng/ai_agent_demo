示例 1：没有 System Prompt
============================================================
这段代码存在**语法错误**，具体取决于你使用的编程语言。从代码风格来看，你很可能是在写 **Python** 代码，但使用了其他语言（如 Ruby 或 JavaScript）的语法。

以下是详细的问题分析和修正方法：

### 1. 主要问题：缺少冒号（Syntax Error）
在 **Python** 中，函数定义的行末尾必须有一个冒号 `:`。

*   **错误代码：**
    ```python
    def add(a, b)  # 这里缺少冒号
        return a + b
    ```
*   **修正：**
    ```python
    def add(a, b):  # 添加冒号
        return a + b
    ```

### 2. 次要问题：缩进（Indentation）
Python 使用缩进来定义代码块（函数体）。虽然你提供的代码看起来有缩进，但在复制粘贴时容易丢失或混用空格/Tab。

*   **规则：** 函数体（`return` 语句）必须相对于 `def` 行进行缩进（通常为 4 个空格）。

---

### 修正后的完整代码（Python 版本）

```python
def add(a, b):
    return a + b

# 测试代码
result = add(3, 5)
print(result)  # 输出: 8
```

---

### 3. 如果你使用的不是 Python

如果你原本打算写其他语言，以下是对应语言的正确写法：

**Ruby:**
Ruby 使用 `end` 关键字来结束代码块，且不需要冒号。
```ruby
def add(a, b)
  return a + b
end
```

**JavaScript:**
JavaScript 使用大括号 `{}` 包裹代码块。
```javascript
function add(a, b) {
    return a + b;
}
```

**Java / C++ / C#:**
这些是静态类型语言，需要指定参数和返回值的类型。
```java
// Java 示例
public int add(int a, int b) {
    return a + b;
}
```

### 总结
如果你是在写 **Python**，请务必在 `def add(a, b)` 后面加上 **冒号 `:`**，并确保 `return` 语句有正确的缩进。

示例 3：Agent 风格的 System Prompt
============================================================
<thinking>
用户想要一个函数来统计字符串中每个字符出现的次数。这是一个常见的编程任务，我需要：
1. 分析需求：统计字符串中每个字符的出现频率
2. 制定计划：编写一个函数，遍历字符串，使用字典或类似结构记录每个字符的计数
3. 逐步执行：编写代码并测试
4. 验证结果：确保函数能正确工作

我将使用Python来实现这个功能，因为Python处理字符串和字典很方便。
</thinking>

我来帮你写一个统计字符串中每个字符出现次数的函数。

```python
def count_characters(text):
    """
    统计字符串中每个字符出现的次数
    
    参数:
        text (str): 要统计的字符串
    
    返回:
        dict: 字符到出现次数的映射
    """
    char_count = {}
    
    for char in text:
        if char in char_count:
            char_count[char] += 1
        else:
            char_count[char] = 1
    
    return char_count

# 测试示例
if __name__ == "__main__":
    test_string = "hello world"
    result = count_characters(test_string)
    print(f"字符串: '{test_string}'")
    print("字符统计:")
    for char, count in sorted(result.items()):
        print(f"  '{char}': {count}")
```

这个函数会返回一个字典，其中键是字符，值是该字符在字符串中出现的次数。例如，对于字符串 "hello world"，输出会显示每个字符（包括空格）的出现次数。

如果你想要更简洁的实现，也可以使用Python的 `collections.Counter`：

```python
from collections import Counter

def count_characters(text):
    """使用Counter统计字符出现次数"""
    return dict(Counter(text))
```

两种方法都能正确工作，第一种方法更基础，第二种方法更简洁。
